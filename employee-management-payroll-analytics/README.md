# Employee Management & Payroll Analytics

SQL project demonstrating schema design, sample data, and analytical queries for an employee-payroll system. Ideal for interviews and corporate projects.

## How to run
1. Install MySQL or use SQLite (adjust small syntax parts).
2. Create database (MySQL):
   - `CREATE DATABASE emp_payroll;`
   - `USE emp_payroll;`
3. Run `sql/01_schema.sql` then `sql/02_sample_data.sql`.
4. Run queries from `sql/03_queries.sql`.

## Files
- `sql/01_schema.sql` — table creation
- `sql/02_sample_data.sql` — sample inserts
- `sql/03_queries.sql` — queries and examples
- `docs/ER_diagram.txt` — ASCII ER diagram
- `writeups/cognizant_sql_writeup.txt` — 300-word write-up
